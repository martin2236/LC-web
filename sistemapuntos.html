Sistema de puntos para clientes

Módulos

Administración de Clientes

Cuenta con un módulo de administración de Clientes. Cada Cliente tiene estados: Activo, Inactivo y Suspendido. Los estados se pueden definir previamente.
Se los puede identificar con una foto, un código Qr, documento o deporte. Se guarda de cada cliente los datos personales inherentes. Poseen una seccion en la cual pueden visualizar la cantidad de puntos acumulados.

Puntos y Premios

El sistema cuenta con una administración para los Puntos y Premios. Cada cliente contará con una tarjeta que puede ingresar a visualizar la cantidad de puntos obteniso y por que premio los puede canjear. Irán sumando puntos a medida que consumen.

Control de consumos y Promociones

Una de las finalidades del sistema, es conocer los gustos del cliente para poder proporcionarle una promoción acorde a sus gustos.
Cada punto es otorgado por un consumo generado por el cliente, el cual puede ser cargado de diferentes formas quedando acorde a las necesidades y facilidades.
El sistema esta preparado para trabajar con diferentes puntos de venta, y/o diferentes comercios, generando para cada uno, un listado diferente y grupal. Lleva un control de consumos por comercio. Cada comercio, podra tener sus promociones y sus descuentos dependiendo lo que el administrador del sistema le autorice.

Puntos Diarios, turnos, cierre mensual

Los niveles de usuarios alcanzan al manejo de puntos, se pueden abrir la variedad de turnos que se desee. Cada usuario tiene la posiblidad de agregar / restar.
Se emiten listados para cada tipo de consumo, todos los consumos juntos, o bien por usuario, y/o por cliente. No existen limites de aperturas de turnos ni de terminales asociadas al sistema. Existe el módulo asociado de "Compensación de puntos".
Vale destacar que este módulo, comparado con la caja, deben ser equitativos.

Niveles de Usuarios

Cuenta con diferentes niveles de usuarios, cada usuario podrá acceder a determinadas partes del sistema, y quedará registrado cada movimiento que hagan dentro del sistema.

Avisos de Promociones, cumpleaños y más

Es menester llevar una calida y afectuosa comunicasion con nustros clientes. Es por ello que este módulo es uno de los mas completos. El sistema avisa de quién es el cumpleaños el día actual, y se podrán emitir listados mensuales de cumpleaños. Se les puede hacer invitaciones para el mes de su cumpleaños y teniendo en cuenta sus gustos, se les podrá hacer una promoción acorde a sus consumos.
Cada promoción nueva, se podrá enviar a todos los clientes o solamente a aquellos que más les guste determinado producto.
Se podrá premiar al que más consuma, o se podrá premiar a quién más interactúe con el sistema de puntos. En pantalla cuando el usuario del sistema lo deseé, podra visualizar que ya fue noticiado de sus beneficios y felicitado por el motivo que fuera; indicando el dia, la hora el motivo y metodo de aviso.

Métodos de interacción con Clientes

Están disponibles 3 métodos de comunicación con los clientes. El primero es mediante el propio sitio web, cuando el cliente se logea, en el panel de bienvenidos, se le pondrá un aviso como importante, el cuál lo podrán eliminar cerrandolo con una cruz (quedará registrado que el usuario ya lo vió, aunque no lo elimine). El segundo es mediante un correo electrónico a la casilla declarada por el cliente. Y el tercero es mediante un SMS a su celular y/o teléfono particular.

Tarjetas y Códigos Qr

El sistema tiene un panel en el cual se genera automaticamente la tarjeta personal del cliente.
Es desición del adquirente del sistema, que método se utilizará para la impresión y entregar dicha tarjeta.
La generación del código Qr, tiene un cifrado de 512 bit y una inserción aleatoria de una numeración temporal para la identificación del Qr y seguridad del sistema.
El cliente con la tarjeta, podrá acceder a sus puntos, sus promociones, y toda aquella comunicación que se le quiera notificar; como ser: premios, salutaciones, promociones y un sin fin de cosas que será de acuerdo a la creación del adquirente del servicio.
Con la contratación del sistema, se entregarán las primeras 100 tarjetas.

Estadísticas

Tenemos la seguridad que las estadísticas son la base para cualquier mejora. Es por ello que tenemos un módulo dedicado especificamente a las estadísticas, las cuales se terminan de adecuar a cada usuario en particular de acuerdo a sus necesidades.
