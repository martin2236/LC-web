Sistema de puntos para clientes

Módulos

Administración de Clientes

Cuenta con un módulo de administración de Clientes. Cada Cliente tiene estados: Activo, Inactivo y Suspendido. Los estados se pueden definir previamente.
Se los puede identificar con una foto, un codigo Qr, documento o deporte. Se guarda de cada cliente los datos personales inherentes. Poseen una seccion en la cual pueden visualizar la cantidad de puntos acumulados.

Puntos y Premios

El sistema cuenta con una administración para los Puntos y Premios. Cada cliente contará con una tarjeta que puede ingresar a visualizar la cantidad de puntos obteniso y por que premio los puede canjear. Irán sumando puntos a medida que consumen.

Control de consumos y Promociones

Una de las finalidades del sistema, es conocer los gustos del cliente para poder proporcionarle una promocion acorde a sus gustos.
Cada punto es otorgado por un consumo generado por el cliente, el cual puede ser cargado de diferentes formas quedando acorde a las necesidades y facilidades.
El sistema esta preparado para trabajar con diferentes puntos de venta, y/o diferentes comercios, generando para cada uno, un listado diferente y grupal. Lleva un control de consumos por comercio. Cada comercio, podra tener sus promociones y sus descuentos dependiendo lo que el administrador del sistema le autorice.

Puntos Diarios, turnos, cierre mensual

Los niveles de usuarios alcanzan al manejo de puntos, se pueden abrir la variedad de turnos que se desee. Cada usuario tiene la posiblidad de agregar / restar.
Se emiten listados para cada tipo de consumo, todos los consumos juntos, o bien por usuario, y/o por cliente. No existen limites de aperturas de turnos ni de terminales asociadas al sistema. Existe el módulo asociado de "Compensación de puntos".
Vale destacar que este módulo, comparado con la caja, deben ser equitativos.

Niveles de Usuarios

Cuenta con diferentes niveles de usuarios, cada usuario podra acceder a determinadas partes del sistema, y quedara registrado cada movimiento que hagan dentro del sistema.

Avisos de Promociones, cumpleaños y mas

Es menester llevar una calida y afectuosa comunicasion con nustros clientes. Es por ello que este modulo es uno de los mas completos. El sistema avisa de quien es el cumpleaños el dia actual, y se podran emitir listados mensuales de cumpleaños. Se les puede hacer invitaciones para el mes de su cumpleaños y teniendo en cuenta sus gustos, se les podra hacer una promocion acorde a sus consumos.
Cada promocion nueva, se podra enviar a todos los clientes o solamente a aquellos que mas les guste determinado producto.
Se podra premiar al que mas consuma, o se podra premiar a quien mas interactue con el sistema de puntos. En pantalla cuando el usuario del sistema lo desee, podra visualizar que ya fue noticiado de sus beneficios y felicitado por el motivo que fuera; indicando el dia, la hora el motivo y metodo de aviso.

Metodos de interaccion con Clientes

Estan disponibles 3 metodos de comunicacion con los clientes. El primero es mediante el propo sitio web, cuando el cliente se logee, en el panel de bienvenidos, se le pondra un aviso como importante, el cual lo podran eliminar cerrandolo con una cruz (quedara registrado que el usuario ya lo vio, aunque no lo elimine). El segundo es mediante un correo electronico a la casilla declarada por el cliente. Y el tercero es mediante un SMS a su celular y/o telefono particular.

Tarjetas y Codigos Qr

El sistema tiene un panel en el cual se genera automaticamente la tarjeta personal del cliente.
Es desicion del adquirente del sistema, que metodo se utilizara para la impresion y entregar dicha tarjeta.
La generacion del codigo Qr, tiene un cifrado de 512 bit y una insercion aleatoria de una numeracion temporal para la identificacion del Qr y seguridad del sistema.
El cliente con la tarjeta, podra acceder a sus puntos, sus promociones, y toda aquella comunicacion que se le quiera notificar; como ser: premios, salutaciones, promociones y un sin fin de cosas que sera de acuerdo a la creacion del adquirente del servicio.
Con la contratacion del sistema, se entregaran las primeras 100 tarjetas.

Estadisticas

Tenemos la seguridad que las estadisticas son la base para cualquier mejora. Es por ello que tenemos un módulo dedicado especificamente a las estadisticas, las cuales se terminan de adecuar a cada usuario en particular de acuerdo a sus necesidades.
